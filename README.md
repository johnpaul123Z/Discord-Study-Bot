Discord Study Bot.

A friendly and interactive study bot for Discord! This bot posts multiple-choice questions to your server's #general channel. When you reply with your answer (A, B, or C), the bot instantly lets you know if you're right or wrongâ€”and if you're wrong, it even provides the correct answer with an explanation.

Features:

Interactive Learning: Engages your server with periodic study questions.
Instant Feedback: Receives immediate responses indicating whether your answer is correct.
Educational Insights: If your answer is incorrect, the bot explains the correct answer.
Simple Setup: Easy to run and deploy.
Getting Started
Prerequisites
Python 3.7+
Discord Bot Token:
Obtain your API keys from the Discord Developer Portal.
